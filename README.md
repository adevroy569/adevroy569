## 🌍 About Me

I'm a **GIS Analyst** with a Master's in **Geo-Data Sciences** from Purdue University, passionate about turning messy geospatial data into clear, reproducible insights. My work spans spatial analysis, cartography, and open geospatial education.

- 🗺️ Currently: **Graduate Assistant in GIS** at Purdue University Libraries
- 📊 Analyzed food access across **1,500+ Indiana census tracts** using Census, USDA, and OpenStreetMap data
- 📚 Authored **6 open-source training modules** for an NSF-funded geospatial cybertraining project
- 🎤 Taught QGIS, remote sensing, and Python at a statewide 2-day GIS workshop
- 🌀 Built a 45-year global tropical cyclone intensity analysis pipeline (NOAA IBTrACS + HURSAT)
- 🌱 Currently exploring: **Survey123** & **ArcGIS Experience Builder**
